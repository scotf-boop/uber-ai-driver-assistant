Uber AI Cloud V15 Professional

第4步：在 PowerShell 复制：
pip install -r requirements.txt

第5步：双击 start_windows10.bat

第6步：浏览器打开：
http://127.0.0.1:5000

V15升级内容：
1. 保留V14的收入/英里/时间提取逻辑。
2. 新增自动计算 $/mile。
3. 新增自动计算 $/hour。
4. 新增订单评级：垃圾单、普通单、好单、优秀单。
5. 新增GPS到接客点成本输入，可计算真实净收入。
6. 新增接单建议：接、谨慎接、不建议接。
7. 新增手动粘贴OCR文字兜底：如果图片OCR有误，可以把识别文字粘贴进去再次分析。

注意：图片OCR需要电脑安装 Tesseract OCR。如果你之前V14能识别，这个版本也可以继续用。
